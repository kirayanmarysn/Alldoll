# Telegram Video Bot

## Setup Instructions

1. **Clone the Repository**  
   Clone this repository to your local machine using the following command:
   ```bash
   git clone https://github.com/kirayanmarysn/Alldoll.git
   ```

2. **Install Dependencies**  
   Navigate into the project directory and install the necessary dependencies:
   ```bash
   cd Alldoll
   npm install
   ```

3. **Configure Bot Token**  
   Obtain your bot token from the [BotFather](https://core.telegram.org/bots#botfather) and add it to a `.env` file in the root of the project:
   ```plaintext
   BOT_TOKEN=your_bot_token_here
   ```

4. **Run the Bot**  
   Start the bot with:
   ```bash
   npm start
   ```

5. **Usage**  
   Follow the instructions in the console to interact with the bot.
